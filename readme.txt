Blendet am 21.03. einen Protestbanner gegen die EU-Urheberrechtsreform ein

== Description ==
Am 21. März 2019 wird die deutschsprachige Wikipedia aus Protest gegen die geplante EU-Urheberrechtsreform 24 Stunden lang nicht erreichbar sein. Mit diesem Plugin kannst du dich dem Protest anschließen!

**Hinweis:** Die Protestseite muss nach dem Installieren unter *Einstellungen* > *SaveYourInternet Protestseite* > **Am Protest teilnehmen** aktiviert werden!

Die EU-Urheberrechtsreform gefährdet das Internet in seiner jetzigen Form. Uploadfilter und Leistungsschutzrecht drohen das Internet kaputtzumachen. Menschen werden schon im Alltag Schwierigkeiten bekommen, beispielsweise beim Diskutieren von Nachrichten oder beim Äußern ihrer Meinung im Netz. Unsere Freiheit zur Teilhabe einzuschränken, um den Sonderinteressen großer Medienkonzerne zu dienen, ist nicht akzeptabel!

Am 23. März werden europaweit Demonstrationen gegen die Reform stattfinden.

== Installation ==
1. Plugin auf deiner Wordpress-Seite installieren
2. Unter *Einstellungen > SaveYourInternet Protestseite* **Am Protest teilnehmen** aktivieren.
3. Bei Bedarf weitere Einstellungen wie das Datum oder die Texte ändern
4. Am ausgewählten Datum wird dann deine Seite von einer InfoBox überlagert.

Unter **Ausschließen** können kommagetrennte Page/Post-IDs eingetragen werden. Diese Seiten/Beiträge sind dann nicht betroffen und können weiterhin aufgerufen werden.
